命名空间

介绍

为了更好地组织类，Ve 提供了包机制，用于区别类名的命名空间。

包

包语句的语法格式为：

use 包名;
use 包名.类名 实例名;

示例:

package nc{
    class classA
    {
        do(){

        }
    }
}
// 其它文件代码文件引用
use nc;
use nc.classA
