变量声明

def 声明

我们使用 def 关键词来声明变量：

def a = 10;

const 声明

const 声明是声明变量的另一种方式，被赋值后不能再改变。

const yearDaysNum = 365;
print(yearDaysNum);//365
yearDaysNum = 366;// error...
